# 中国近现代史纲要 · 答题练习网页

一个基于 HTML/CSS/JavaScript 的近代史题库练习工具，支持多种刷题模式、错题本和本地进度保存，无需服务器即可在浏览器中运行。

## ✨ 功能特点

- **多种做题模式**  
  📋 顺序做题 | 🎲 随机选题 | 🆕 未做题模式 | 📕 错题本模式 | 🔢 从指定题号开始

- **交互功能**  
  ✔ 选择选项后提交，自动判定对错  
  🔍 查看答案功能（不记录成绩）  
  ◀ 上一题 / 下一题 ▶ 自由切换

- **视觉反馈**  
  🟢 正确选项显示为绿色  
  🔴 错误选项显示为红色，同时高亮正确答案

- **错题本**  
  自动记录答错的题目，可随时进入错题本模式集中复习

- **进度保存**  
  使用 `localStorage` 自动保存做题记录和错题本，刷新或关闭浏览器后仍可继续

## 🚀 快速开始

用浏览器直接打开 `index.html` 即可使用。
或者保存刷题链接(因本人比较懒，所以只部署到了github上)

## 📖 使用说明

### 模式切换
点击顶部的模式按钮（顺序/随机/未做/错题本/从题号），然后点击 **✅ 应用** 按钮生效。  
- **随机模式**：需在输入框填写抽取的题目数量。  
- **从题号模式**：需输入起始题号（从1开始）。  

### 答题操作
- 点击选项选中，再点击 **📥 提交** 完成作答。  
- **🔍 查答案** 仅显示正确答案，不会记录为已做，也不会影响错题本。  
- 使用 **◀ 上一题** / **下一题 ▶** 切换题目。

### 错题本
- 答错的题目自动加入错题本。  
- 点击 **📕 错题本** 模式，然后应用，即可只复习错题。  
- 错题再次答对后会自动从错题本移除。

### 重置进度
点击 **🔄 重置进度** 可清除所有做题记录和错题本。

## 🛠 技术实现

- 纯前端实现，无任何依赖。
- 使用 `localStorage` 进行数据持久化。
- 响应式布局，支持手机和电脑。
